# Windows 12 Nova Edition

Браузерная концепция операционной системы с окнами, файловой системой, встроенным браузером и мини-играми.

## Быстрый старт локально

```bash
python -m http.server 8000
```
Откройте `http://localhost:8000`.

## Деплой на GitHub Pages (для Repchic/OS)

1. Убедитесь, что файлы находятся в корне репозитория.
2. В GitHub откройте **Settings → Pages**.
3. В **Build and deployment** выберите **Deploy from a branch**.
4. Укажите ветку `main` и папку `/root`.
5. После сохранения GitHub выдаст публичную ссылку.

> При необходимости можно создать ветку `gh-pages` и использовать её для публикации.

## Публикация изменений в GitHub

Если вы видите файлы локально, но не видите их в GitHub, нужно отправить изменения в удалённый репозиторий.

```bash
git status -sb
git push origin main
```
